# 🧀cheese-pot
일본 드라마, 영화, 애니메이션에서 나온 명소들을 찾아가는 서비스
> 요즘 영화, 드라마, 애니메이션  등 일본 문화콘텐츠를 좋아하는 마니아들이 늘고 있는 추세입니다. 내가 좋아하는 콘텐츠에 나오는 장소를 직접 가보고 싶지 않나요? 치즈팟으로 와요!
